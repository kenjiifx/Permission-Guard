# Demo Assets

This directory stores marketing/demo media used in the top-level `README.md`.

## Required demo file

- `demo-scan.gif`

Recommended capture flow:

1. Run:
   - `npx @kenjiifx/permissionguard scan examples/bad-policy.json`
2. Record terminal output as a short GIF (8-15 seconds)
3. Save file as:
   - `assets/demo-scan.gif`
